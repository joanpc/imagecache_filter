Image Cache Filter Module

Drupal module to use image Cache presets on inline images.

------------
Installation
------------

Copy to your sites/all/modules and enable it.

Enable it on your input format at admin/settings/filters, Select a imagecache preset on the Configuration tab and
Rearange it just before the HTML corrector.
 

 
know bugs: It rewrites images already using an imagecache preset.
